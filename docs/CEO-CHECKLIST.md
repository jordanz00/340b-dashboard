# CEO Evaluation Checklist — HAP 340B Advocacy Dashboard

Use this checklist before presenting the dashboard to executives, lawmakers, or hospital CEOs.

---

## First impression

When a CEO opens this dashboard, they should see:

- [ ] **Clear headline** — "340B Drug Pricing Program" is immediately visible and understandable
- [ ] **Professional design** — Clean layout, readable typography, no clutter
- [ ] **HAP branding** — Logo and organization name are visible and professional

---

## Core content

- [ ] **Map shows state protections at a glance** — Blue = protection, gray = no protection; legend is clear
- [ ] **Key metrics are prominent** — 72 PA hospitals, $7.95B community benefits, 7% market share, 179 HRSA audits
- [ ] **HAP Position is clear** — Protect 340B integrity and contract pharmacy access
- [ ] **Data sections are readable** — Key findings, executive strip, community benefit, access, PA safeguards

---

## Navigation and usability

- [ ] **State selection works** — Click a state on the map; details appear
- [ ] **Filters work** — All / Protection / No protection toggle correctly
- [ ] **Share link works** — Copies a URL that preserves state context
- [ ] **Simple navigation** — No confusing menus or hidden features

---

## Export and presentation

- [ ] **Print / PDF button works** — Opens a polished print view in a new tab
- [ ] **Download PDF (image) works** — Produces a 3-page PDF for offline sharing
- [ ] **Print preview looks professional** — No blank pages, map visible, KPIs show final numbers
- [ ] **Pennsylvania default** — When no state is selected, print/PDF shows PA context

---

## Quality bar

If any of the above fail, the dashboard needs improvement before a CEO presentation.

Run `python3 dashboard-audit.py` and manually test map, filters, share, Print/PDF, and Download PDF before release.
